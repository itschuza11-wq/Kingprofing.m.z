# Kingprofit (Next.js + Tailwind + Supabase)

This is a ready-to-deploy project for Kingprofit.

## Quick start
1. Create a repository on GitHub and upload the contents of this zip.
2. Create a `.env.local` file at the project root (or set Vercel environment variables).
3. Install and run locally:
   ```bash
   npm install
   cp .env.example .env.local
   npm run dev
   ```
4. Deploy on Vercel (link your GitHub repo). In Vercel, set the same env vars under Project → Settings → Environment Variables.
